# voice-translate
한국어 음성인식 5개국어 번역기

#	기능	상태
1	🎙️ 실시간 음성인식 (한국어)	
2	🌐 5개국어 동시 번역 (EN/ZH/JA/FR/DE)	
3	🔊 TTS 발음 재생 (언어별 원어민 발음)	
4	🕐 번역 히스토리 (최대 50개)	
5	⭐ 즐겨찾기 (localStorage 영구 저장)	
6	📋 클립보드 복사 (언어별 원클릭)	
7	🎭 감정/톤 감지 (긍정/부정/중립 배지)	
8	📖 단어 카드 (플립 플래시카드)	
9	🎛️ TTS 속도/피치 조절 (슬라이더)	
10	⌨️ 타이핑 모드 (텍스트 직접 입력)	
11	💡 문화 팁 (언어별 국가 문화 힌트)	
12	🌙 다크/라이트 모드 토글	
13	🔑 로마자 표시 placeholder (중국어/일본어)	
14	🔗 URL 공유 기능
